- DIY seat cover creation using blankets
- Simple sewing and fitting approach
- Documentation of the process with images

## Getting Started

### Prerequisites

- Sewing kit or machine

1. Download or clone the repository:

```bash
git clone https://github.com/justin-napolitano/seat-covers.git
```

2. Extract the images.zip file to view the reference images:
